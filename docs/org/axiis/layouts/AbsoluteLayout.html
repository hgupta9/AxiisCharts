Packageorg.axiis.layouts
Classpublic class AbsoluteLayout
InheritanceAbsoluteLayout BaseLayout AbstractLayout flash.events.EventDispatcher
Implements ILayout

AbsoluteLayout is a layout which does not have reference geometries. AbsoluteLayouts can be used when you want to produce visualizations that has its drawingGeometries positioned without respect to an references.
