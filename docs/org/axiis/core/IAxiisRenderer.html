Packageorg.axiis.core
Interfacepublic interface IAxiisRenderer
Implementors DataTip, TextDataTipContent

An interface for objects that are used to render an item from the Axiis render cycle. During the render cycle, the layouts update their currentLabel, currentDatum, currentValue, and currentIndex properties. Data tip content renderers are expected to implement this interface.
